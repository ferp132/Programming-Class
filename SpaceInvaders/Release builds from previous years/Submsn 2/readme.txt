Space Invaders Readme - Jack Wilson

In Menus:
+ Scroll through menus using the arrow keys (up/down)
+ Select a menu item with the enter key

While Playing:
+ Move the ship with the arrow keys (left/right)
+ Fire with the spacebar
+ Destroy all of the aliens before they reach your bunkers
+ A mystery ship will occasionally fly across the top of the screen - this grants extra points!
